# _claude/ — AI Quarantine

This folder holds AI-generated drafts, summaries, and scratch work. **Content here is not canonical.**

## Rules

- AI writes here freely — drafts of emails, meeting recaps, QBR outlines, code snippets, anything.
- **Only Dan promotes content** from here into the canonical folders (`20-Accounts/`, `50-Meetings/`, etc.). AI should not self-promote.
- Filename convention: `YYYY-MM-DD-short-slug.md` (e.g., `2026-04-17-gev-qbr-outline.md`)
- Tidy this folder during weekly reviews. Promote what's useful, delete what's not.
- Nothing lives here long-term.

## Why

Your authentic thinking stays in the main folders. AI output stays visible but separate so the graph doesn't get polluted and so it's always clear what you wrote vs. what was generated.
