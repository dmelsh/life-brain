# Life Brain

Dan's personal second brain. Paired with OpenClaw Home CoS.

## Quick start

- **Today's note**: `10-Daily/2026/` → create `YYYY-MM-DD.md` from `_templates/daily-note.md`
- **New project?** Duplicate `_templates/project-note.md` into `30-Projects/`
- **Something ongoing with no end date?** That's an Area. Go to `40-Areas/`
- **Looking for something?** Start at a MOC in `80-MOCs/`.

## For AI sessions

Read `CLAUDE.md` first.

## Structure at a glance

| Folder | What goes here |
|---|---|
| `00-Inbox/` | Unprocessed capture |
| `10-Daily/` | Daily notes |
| `20-People/` | Family, friends, contractors, tenants |
| `30-Projects/` | Active projects (have outcomes and end dates) |
| `40-Areas/` | Ongoing life areas (Home, Rental, Fitness, Family, Finances) |
| `50-Resources/` | Reference material |
| `60-Journal/` | Personal reflections, weekly reviews |
| `70-Travel/` | Trip planning and logs |
| `80-MOCs/` | Topic indexes |
| `90-Archive/` | Completed/inactive |
